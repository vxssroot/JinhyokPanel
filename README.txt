HOW TO DEPLOY:
--------------
1. Install XAMPP or any LAMP stack
2. Copy the entire folder to htdocs (e.g., C:\xampp\htdocs\jinhyok-panel)
3. Import setup.sql into your MySQL database
4. Update config.php with your database credentials
5. Visit http://localhost/jinhyok-panel/index.php
